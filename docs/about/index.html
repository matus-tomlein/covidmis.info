Fact checks related to COVID-19.

CovidMis.info aggregates fact checks related to COVID-19 from multiple fact checking sites. It provides statistics, links between fact checks and links of fact checks to articles found on the Web.

To learn more about the project, check out CovidMis.info: Exploration of COVID-19 Fact Checks.

Slovak version is available at CovidDez.info.
